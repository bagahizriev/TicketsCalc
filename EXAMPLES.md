# Примеры использования

## Пример 1: Базовый расчет

### Входные данные:

- Бюджет: 10000
- Показы: 50000
- Переходы: 500

### Результаты:

- eCPC: 20.00 ₽
- eCPM: 200.00 ₽
- CTR: 1.000 %

### Формулы:

```
eCPC = 10000 / 500 = 20.00 ₽
eCPM = (10000 / 50000) × 1000 = 200.00 ₽
CTR = (500 / 50000) × 100 = 1.000 %
```

## Пример 2: Большие числа

### Входные данные:

- Бюджет: 1 000 000
- Показы: 5 000 000
- Переходы: 25 000

### Результаты:

- eCPC: 40.00 ₽
- eCPM: 200.00 ₽
- CTR: 0.500 %

### Копирование всех результатов:

```
Потрачено: 1 000 000.00 ₽, Показы: 5 000 000, Переходы: 25 000, CTR: 0.500 %, eCPC: 40.00 ₽, eCPM: 200.00 ₽
```

## Пример 3: Дробные значения

### Входные данные:

- Бюджет: 1234.56
- Показы: 10000
- Переходы: 123

### Результаты:

- eCPC: 10.03 ₽ (обрезано, не округлено)
- eCPM: 123.45 ₽ (обрезано, не округлено)
- CTR: 1.230 % (округлено математически)

### Особенности:

- eCPC: 1234.56 / 123 = 10.037... → обрезается до 10.03
- eCPM: (1234.56 / 10000) × 1000 = 123.456 → обрезается до 123.45
- CTR: (123 / 10000) × 100 = 1.23 → округляется до 1.230

## Пример 4: Малые значения CTR

### Входные данные:

- Бюджет: 5000
- Показы: 1000000
- Переходы: 125

### Результаты:

- eCPC: 40.00 ₽
- eCPM: 5.00 ₽
- CTR: 0.013 %

## Использование приложения

### Ввод данных

1. Введите бюджет (можно с точкой для копеек)
2. Введите количество показов (целое число)
3. Введите количество переходов (целое число)
4. Результаты обновляются автоматически

### Копирование результатов

#### Отдельный результат:

- Нажмите на иконку копирования рядом с нужной метрикой
- Скопируется: "20.00 ₽" или "1.000 %"

#### Все результаты:

- Нажмите кнопку "Копировать все результаты"
- Скопируется полная строка со всеми данными

### Очистка

- Нажмите кнопку "Очистить" для сброса всех полей

### Переключение темы

- Нажмите иконку солнца/луны в правом верхнем углу
- Тема сохраняется в localStorage

## Валидация ввода

### Разрешенные символы:

- Цифры: 0-9
- Точка: . (для десятичных дробей в бюджете)
- Запятая: , (автоматически заменяется на точку)

### Автоматически удаляются:

- Пробелы
- Разделители разрядов (\_, ', `)
- Буквы и спецсимволы

### Примеры ввода:

- `1 000 000` → `1000000`
- `1,234.56` → `1234.56`
- `1_000_000` → `1000000`

## Форматирование вывода

### Целые числа:

- `1000` → `1 000`
- `1000000` → `1 000 000`

### Дробные числа:

- `1234.56` → `1 234.56`
- `1000000.12` → `1 000 000.12`

## Установка как PWA

### iOS (Safari):

1. Откройте приложение в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Прокрутите вниз и выберите "На экран «Домой»"
4. Нажмите "Добавить"
5. Приложение появится на домашнем экране

### Android (Chrome):

1. Откройте приложение в Chrome
2. Нажмите на три точки в правом верхнем углу
3. Выберите "Установить приложение" или "Добавить на главный экран"
4. Подтвердите установку
5. Приложение появится в списке приложений

### Desktop (Chrome/Edge):

1. Откройте приложение
2. В адресной строке появится иконка установки (⊕)
3. Нажмите на нее
4. Подтвердите установку
5. Приложение откроется в отдельном окне

## Офлайн режим

После первого посещения приложение работает без интернета:

- Все функции калькулятора доступны
- Сохраняется выбранная тема
- При отсутствии соединения показывается страница "Вы офлайн"

## Haptic Feedback (iOS)

Вибрация при:

- Фокусе на поле ввода (легкая)
- Копировании результата (средняя)
- Очистке полей (легкая)
- Переключении темы (легкая)

## Горячие клавиши

- `Tab` - переход между полями
- `Enter` - переход к следующему полю
- `Escape` - снятие фокуса
- `Ctrl/Cmd + A` - выделить все в поле

## Доступность

- Все кнопки имеют минимальный размер 44x44px
- Поддержка клавиатурной навигации
- ARIA-метки для screen readers
- Высокий контраст текста (WCAG AA)
- Фокус-индикаторы для всех интерактивных элементов
