## 📋 Что дальше?

### 1️⃣ Локальный запуск (для разработки)

```bash
npm run dev
```

Откройте http://localhost:3000 в браузере.

### 2️⃣ Деплой на GitHub Pages

#### Вариант A: GitHub Desktop (рекомендуется)

1. Откройте GitHub Desktop
2. File → Add Local Repository
3. Выберите папку проекта
4. Publish repository → назовите `TicketsCalc`
5. Перейдите на github.com в ваш репозиторий
6. Settings → Pages → Source: "GitHub Actions"
7. Готово! Через 2-3 минуты приложение будет доступно

#### Вариант B: Командная строка

```bash
# 1. Создайте репозиторий на github.com с именем TicketsCalc

# 2. Выполните команды:
git init
git add .
git commit -m "Initial commit: PWA калькулятор билетов"
git branch -M main
git remote add origin https://github.com/ВАШ_USERNAME/TicketsCalc.git
git push -u origin main

# 3. Перейдите в Settings → Pages → Source: "GitHub Actions"
```

### 3️⃣ Ваше приложение будет доступно по адресу:

```
https://ВАШ_USERNAME.github.io/TicketsCalc/
```

## 📱 Установка как приложение

### iOS (Safari)

1. Откройте приложение в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Выберите "На экран «Домой»"
4. Нажмите "Добавить"

### Android (Chrome)

1. Откройте приложение в Chrome
2. Нажмите на три точки
3. Выберите "Установить приложение"

### Desktop (Chrome/Edge)

1. Откройте приложение
2. Нажмите на иконку установки в адресной строке
