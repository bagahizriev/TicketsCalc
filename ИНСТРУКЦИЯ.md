# 📱 Калькулятор билетов - Инструкция

## 🚀 Быстрый старт

### Локальный запуск

```bash
npm run dev
```

Откройте http://localhost:3000

### Деплой на GitHub Pages

1. Создайте репозиторий `TicketsCalc` на github.com
2. Выполните:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_USERNAME/TicketsCalc.git
git push -u origin main
```

3. Settings → Pages → Source: "GitHub Actions"
4. Готово! Приложение будет на `https://ВАШ_USERNAME.github.io/TicketsCalc/`

## 📊 Как пользоваться

1. Введите **Бюджет** (например: 10000)
2. Введите **Показы** (например: 50000)
3. Введите **Переходы** (например: 500)
4. Результаты обновятся автоматически

### Результаты:

- **eCPC** - стоимость одного клика
- **eCPM** - стоимость 1000 показов
- **CTR** - процент кликабельности

### Кнопки:

- 📋 Копировать отдельный результат
- 📄 Копировать все результаты
- 🗑️ Очистить все поля
- 🌓 Переключить тему

## 🎯 Формулы расчета

```
eCPC = Бюджет / Переходы
eCPM = (Бюджет / Показы) × 1000
CTR = (Переходы / Показы) × 100%
```

### Пример:

- Бюджет: 10 000 ₽
- Показы: 50 000
- Переходы: 500

**Результат:**

- eCPC: 20.00 ₽
- eCPM: 200.00 ₽
- CTR: 1.000 %

## 💡 Советы

1. Приложение сохраняет выбранную тему
2. Можно копировать каждый результат отдельно
3. Кнопка "Копировать все" создает полную строку с данными
4. Работает на всех устройствах: телефон, планшет, компьютер
5. После установки работает без интернета
